# Système de Gestion MSDA 🎓
Application d'interfaçage Python-SQL pour la gestion des étudiants et des notes.

## Fonctionnalités
* Authentification sécurisée.
* Gestion CRUD (Ajout, Modification, Suppression) des étudiants et des notes.
* Visualisation des données avec Plotly.
* Historique des actions (Logs) en base de données.
* Export des données en CSV.

## Installation
1. Configurer la base de données MySQL avec le script fourni.
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer l'app : `streamlit run app_streamlit.py`
